# Vanilla

A sample app in vanilla TypeScript that showcases authentication, datastore and storage usage.

## Getting started

Make sure you have [node.js](https://nodejs.org) LTS installed.

```bash
git clone https://github.com/buildwithjuno/examples
cd vanilla
npm ci
```

## Local development

```
npm run dev
```
